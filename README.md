# Customer-Behaviour-Shopping-Project.
This project analyzes customer shopping data to understand buying patterns and spending behavior.The goal is to identify useful insights that can help improve sales and marketing strategight.
Objectives : Study customer purchasing behavior, Identify top-selling products, Analyze spending patterns, Perform basic customer segmentation.
Dataset : Customer ID, Age, Gender, Product Category, Purchase Amount, Payment Method, Purchase Date.
Tools Used : Excel, SQL , Power BI. 
Analysis Performed : Data cleaning (removed duplicates, handled missing values), Age and gender analysis, Sales trend analysis, Product category analysis.
Key Insights : Young adults spend more compared to other age groups, Some product categories generate higher revenue, Weekend sales are higher than weekdays.
Conclusion : This project helps understand customer behavior and provides insights to improve business performance.
